Intranet

Capabilities

Embed the viewer and interact with model data through a structured API.

iFrame Embed

Embed the full IFC Viewer into any web application via a responsive iFrame. Supports resize, fullscreen, and cross-origin messaging.

Model Registration API

Register and manage IFC models programmatically. Upload, update, and delete models through a REST API.

Model Query API

Query IFC properties, elements, and spatial structure. Filter by IfcClass, PropertySet, or GlobalId. Returns structured JSON.

Visual Markup API

Color elements by GlobalId, highlight selections, and apply visual markups programmatically. Drive the viewer state from your app.

Technical Specifications

  • REST API — Standard HTTP endpoints
  • BCF 2.1 & 3.0 — Issue management
  • GlobalId coloring — Element-level control
  • iFrame — Cross-origin messaging
  • IFC 2x3 & IFC4 — Full format support
  • JSON responses — Structured data
