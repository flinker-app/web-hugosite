SDK

IFC Viewer SDK

Embed powerful 3D IFC viewing capabilities into your web applications. High-performance WebGL rendering that handles large construction models with ease.

Documentation Live Demo

Key Features

High Performance

WebGL-based rendering optimized for large IFC models. Smooth navigation even with millions of elements.

Measurement Tools

Built-in distance, area, and angle measurements. Section planes for interior views.

BCF Support

Create and manage BCF issues directly in the viewer. Compatible with BCF 2.1 and 3.0.

Model Tree

Hierarchical model browser with search, filtering, and selection capabilities.

Property Panels

Display IFC properties, quantities, and custom attributes for selected elements.

Customizable

Full control over UI, colors, and behavior. Works with React, Angular, Vue, or vanilla JS.

Supported Formats

  • IFC 2x3 - Industry standard
  • IFC4 - Latest specification
  • IFC4x3 - Infrastructure extension

Integration Options

  • npm package - Easy installation
  • CDN - Quick prototyping
  • iframe embed - Simplest integration
